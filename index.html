Dual-Venc (DV) 4D Flow MRI Acquisition & Reconstruction

This velocity encoding strategy increases the sensitivity to slow flow while maintaining a high dynamic range of velocities that can be measured. The downside of this method is it requires longer scan times. To overcome this addition, I compared two radial trajectories a in vitro for DV cardiac 4D flow imaging: 3D radial (PC-VIPR) versus stack of stars radial (PC-SOS), with benchtop particle imaging velocimetry serving as a reference standard.

  • Phase correction code (4D FFT for unwrapping - C++)
  • Recon code (flow-encoding [Private] - C++)
  • Post-processing code (validation - Matlab)
  • Validation paper
